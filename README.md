# Fluke289-sigrok-cli-GUI
A simple Python script that will read data over original IR to UART cable. 

Prerequisits are that you have sigrok-cli installed and selected in the settings tab, one can also enable CSV logging from there. 

dependancies -> pip install PySide6 pyserial

Simply hoook up the original (or DIY) cable, select com port and click start - enjoy!

Feel free to use and change any aspect of the script - fair use only!
